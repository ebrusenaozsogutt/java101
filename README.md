# java101

Bu repo, Java öğrenirken yazdığım küçük kod örneklerini içerir.  
Örnekler arasında çift/tek sayı bulma, döngü ve koşul uygulamaları gibi temel Java programları bulunuyor.

## İçerik
- Temel Java programlama örnekleri
- Koşul ve döngü uygulamaları
- Konsol tabanlı mini projeler

## Amaç
Java dilini öğrenmek ve temel programlama mantığını pekiştirmek.
